# C1 Exercise Materials: Literature Review & Synthesis

These materials support the exercises in [Module C1: Literature Review & Synthesis](../../modules/c1-lit-review.qmd).

## Contents

### Citation Verification

- **`citation_verification_table.md`** — Structured worksheet for the "test the anti-pattern" exercise (Part 5). Students ask AI for citations, then verify each one in Google Scholar.

### Curated Paper Sets

These folders contain verified, real citations that students can use as a reference set for the AI-assisted synthesis exercise (Parts 2-4). Each set includes 4-5 well-known papers with full citations, one-sentence summaries, and identification strategies.

- **`curated_papers/teacher_quality.md`** — Papers on teacher quality and student outcomes
- **`curated_papers/immigration_wages.md`** — Papers on immigration and wages
- **`curated_papers/microfinance.md`** — Papers on microfinance impacts

## How to Use

1. **Parts 1-4** (the recommended workflow): Students pick a topic, brainstorm search terms with AI, find real papers themselves (or use the curated sets as a starting point), then use AI to synthesize.

2. **Part 5** (the anti-pattern): Students use the `citation_verification_table.md` to test what happens when they ask AI to generate citations directly.

3. **Curated paper sets** can be used as:
   - A shortcut for students who don't have time for the full database search (Parts 2-3)
   - A verification benchmark — students can compare AI-generated citation lists against these known-good references
   - Abstracts to paste into AI for the synthesis exercise (Part 3)

## Note

The curated paper sets contain verified real citations as of March 2026. If using these materials in future semesters, spot-check that citation details are still accurate (e.g., papers have not been retracted or corrected).
