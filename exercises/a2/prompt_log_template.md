## Instructions

Use this log to track your AI prompting throughout the semester. Each time you use an AI tool for coursework:

1. Record your original prompt and rate the output quality (1 = useless, 5 = exactly what I needed).
2. Identify what was missing using the ROCS framework: **R** (Role/Context), **O** (Objective), **C** (Constraints), **S** (Specifics).
3. Write a revised prompt and rate the improved output.
4. Note what you learned for next time.

Bring this log to class for discussion when your instructor requests it.
